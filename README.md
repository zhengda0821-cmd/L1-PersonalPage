# L1 Personal Page

A sleek single-page personal website featuring a real-time clock and a glassmorphism UI style.

## Live Demo
- GitHub Pages: https://zhengda0821-cmd.github.io/L1-PersonalPage/

## Completed Tasks

### 1. Web Development
- **Main Interface (index.html):** Built a clean single-page layout with a dynamic header, real-time clock, and core sections (About, Skills, Projects).
- **Styling (style.css):** Implemented a glassmorphism design with gradients, blur effects, and hover micro-animations for a premium look.
- **Dynamic Features (script.js):** Added JavaScript for a real-time clock (updates every second) and auto-updated footer year.

### 2. Git & GitHub Integration
- Initialized a local Git repository for the project.
- Committed the website source files and pushed them to GitHub successfully.

### 3. Deployment
- Enabled **GitHub Pages** and deployed the site from the `main` branch (root).

## File Structure
- `index.html` — Core page structure and content
- `style.css` — Visual styling (glassmorphism + responsive layout)
- `script.js` — Real-time clock + small dynamic interactions
- `README.md` — Project report / documentation

## How to Run Locally
1. Download or clone this repository.
2. Open `index.html` in your browser.
   - (Optional) Use VS Code Live Server for a smoother dev experience.
